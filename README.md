#Описание программы
Randomquote - GUI-приложение для генерации случайных цитат. Позволяет сгенерировать цитаты, отобразить историю этих цитат в виде списка и сохранить ее, реализовать фильтрацию цитат по автору и по теме.
##Требования для запуска
- Python 3.10
- Библиотеки: Tkinter

##Как пользоваться
1. При запуске появляется меню:
   - кнопка для генерации случайеых цитат "Случайная цитата"
   - окно вода для фильтра по автору
   - окно ввода для фильтра по теме
   - кнопка для применения указанных фильтров "Применить фильтр"
   - окно для вывода истроии сгенерированных цитат
   - кнопка для сохранения истории сгенерированных цитат "Сохранить историю"
   - кнопка для загрузки истории сгенерированных цитат "Загрузить историю"
2. Введите необходимые данные и используйте нужные кнопки
3. Данные сохраняются в quote_history.json
